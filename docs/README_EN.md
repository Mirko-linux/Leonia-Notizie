# Leonia+ News

**Leonia+ News** is an open-source Telegram bot that, based on generative artificial intelligence, summarizes 5 news items from 5 authoritative sources (ANSA, TGcom24, etc.) in Italian.

## Features

* **Multi-Source Aggregation**: Retrieves news in real time from multiple sources through ethical web scraping.
* **Powered by AI**: Uses advanced models, via the Openrouter.com service, to summarize news.
* **Redis Database**: Manages registered groups via a Key-Value database for maximum persistence.
* **Operating Hours**: Active only during the day (from 6:00 AM to 6:00 PM, Rome time zone).
* **Evening Editorials**: At 6:00 PM, generate an in-depth analysis of your choice using the AI ​​and publish it using the Telegra.ph service.
* **Built-in Ping Service**: Internal Flask server for port-binding to Render.
