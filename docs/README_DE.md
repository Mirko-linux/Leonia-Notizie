# Leonia+ News

**Leonia+ News** ist ein Open-Source-Telegram-Bot, der mithilfe generativer künstlicher Intelligenz fünf Nachrichten aus fünf renommierten Quellen (ANSA, TGcom24 usw.) auf Italienisch zusammenfasst.

## Funktionen

* **Multi-Source-Aggregation**: Ruft Nachrichten in Echtzeit aus verschiedenen Quellen durch ethisches Web-Scraping ab.

* **KI-gestützt**: Nutzt fortschrittliche Modelle des Openrouter.com-Dienstes zur Nachrichtenzusammenfassung.

* **Redis-Datenbank**: Verwaltet registrierte Gruppen über eine Key-Value-Datenbank für maximale Datensicherheit.

* **Betriebszeiten**: Aktiv nur tagsüber (von 6:00 bis 18:00 Uhr römischer Zeit).

* **Abendliche Leitartikel**: Um 18:00 Uhr erstellen Sie mithilfe der KI eine detaillierte Analyse zu einem Thema Ihrer Wahl und veröffentlichen diese über den Telegra.ph-Dienst.

* **Integrierter Ping-Dienst**: Interner Flask-Server für die Portbindung an Render.
